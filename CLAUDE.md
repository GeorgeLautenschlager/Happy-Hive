# Happy-Hive

A GTD-based agentic workflow system built with LangGraph and LangChain.

## Stack

- Python 3.10+
- LangGraph / LangChain
- FastAPI + Uvicorn
- Pydantic v2
- Poetry for dependency management

## Commands

```bash
poetry install          # install dependencies
poetry run pytest       # run tests
poetry run black .      # format
poetry run ruff check . # lint
```
